# ComputerFromScratch

# 持续更新ing

欢迎来到我的 **ComputerFromScratch设计项目**！  

本项目是数字电路学习的实践记录，从最基本的晶体管和门电路开始，希望能一步步构建了一个功能完整的简单电脑，目前完成进度为**拥有十个基本功能的8位算术逻辑单元（ALU）**

接下来我将继续推进时序电路的尝试，欢迎持续关注。

文件内所有模块均使用 **Logisim 仿真软件** 手工搭建，遵循 **自底向上、模块化、层次化** 的设计原则。

---

# 📖 项目简介

本项目目前实现了一个 **8位 ALU**，支持以下操作：

### 算术运算
- 加法（操作码0000）
- 减法（操作码0001）
- 递增（操作码0010）
- 递减（操作码0011）

### 逻辑运算
- 按位与（操作码0100）
- 按位或（操作码（0101）
- 按位异或（操作码0110）
- 取反（操作码0111）

### 移位运算
- 逻辑左移（操作码1000）
- 逻辑右移（操作码1001）

### 比较运算
- 相等比较
- 无符号小于
- 无符号大于

### 特殊操作
- 直通 A
- 清零

ALU 采用 **4位操作码（OP[3:0]）** 选择功能，输出：

- 8位结果 `Result[7:0]`

整个逻辑设计 **完全由门电路搭建**，不依赖 Logisim 内置的高级组件（总线拆分器仅用作整理电路）。

# 🎯 设计目标

### 自底向上
从 **晶体管（场效应管）** 搭建基本门电路：

- AND
- OR
- NOT
- XOR

再用这些门电路构建：

- 半加器
- 全加器
- 多位加法器
- 减法器

最终组合成完整的 **ALU**。

---

### 模块化与层次化

每个功能单元封装为独立子电路：

- 上层模块只通过引脚连接
- 内部实现可随时优化
- 不影响外部接口

---

### 可扩展

ALU预留足够的操作码空间，未来可轻松添加新功能：

- 乘法
- 循环移位
- 更多算术运算
- 相等比较
- 无符号小于
- 无符号大于
- 直通 A
- 清零
- 零标志 `Zero`
- 进位 / 借位标志 `Carry`
- 符号标志 `Negative`
- 溢出标志 `Overflow`
